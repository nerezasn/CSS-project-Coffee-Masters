# ☕ Coffee Masters Order Form

A responsive coffee ordering interface built with HTML and CSS as part of my Frontend Masters learning journey.

This project recreates a modern coffee shop checkout experience, allowing users to review their order, enter customer information, provide payment details, and view the final order summary.
